Introduction to HTML

HTML, stands for HyperText Markup Language

HTML is the foundation of most webpages. It's a language used to structure and display content on the web, defining the content and structure of documents that are displayed in a web browser. HTML uses elements (like <tags> and attributes) to organize text, images, links, and other multimedia elements.

  • The <!DOCTYPE html> declaration defines that this document is an HTML5 document

  • The <head> element contains meta information about the HTML page

  • The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)

  • The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

  • The <h1> element defines a large heading

  • The <p> element defines a paragraph

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
</body>
</html>

Tags in HTML

HTML tags are the building blocks of web pages, providing instructions to browsers on how to display content.

They're enclosed in angle brackets < > and typically come in pairs (opening and closing). Some common examples include <h1> for headings, <p> for paragraphs, <a> for links, and <img> for images.

Element in HTML

In HTML, an element is a building block of a web page that defines a section or a component of the page's structure and content. It's composed of a start tag, content (which can include other elements or text), and an end tag.

<h1> is a Heading Element</h1>
<p>This is a paragraph Element</p>

Void Element

Some HTML elements, like <br>, <hr>, <img>, and <input>, are self-closing and don't have an end tag, as they represent an event or a visual element.

Heading Element

HTML headings are titles or subtitles that you want to display on a webpage.

HTML headings are defined with the <h1> to <h6> tags.

<h1> defines the most important heading. <h6> defines the least important heading.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
